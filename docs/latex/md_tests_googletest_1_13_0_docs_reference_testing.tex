Defines an individual test named $\ast${\ttfamily Test\+Name}$\ast$ in the test suite $\ast${\ttfamily Test\+Suite\+Name}$\ast$, consisting of the given statements.

{\ttfamily GTEST\+\_\+\+SKIP()}

Prevents further test execution at runtime.

Can be used in individual test cases or in the {\ttfamily Set\+Up()} methods of test environments or test fixtures (classes derived from the \mbox{\hyperlink{md_tests_googletest_1_13_0_docs_reference_testing_Environment}{\`{}\+Environment\`{}}} or \mbox{\hyperlink{md_tests_googletest_1_13_0_docs_reference_testing_Test}{\`{}\+Test\`{}}} classes). If used in a global test environment {\ttfamily Set\+Up()} method, it skips all tests in the test program. If used in a test fixture {\ttfamily Set\+Up()} method, it skips all tests in the corresponding test suite.

Similar to assertions, {\ttfamily GTEST\+\_\+\+SKIP} allows streaming a custom message into it.
